# Proveably Random Raffle Contracts

## About

This code is to create a provably random smart contract lottery.

## What we want it to do?

1. Users shoule be able to enter the raffle by paying fo a ticket. The ticket fees are going to be the prize the winner receives.
2. The Lettery should be automatically and programmatically draw a winner after a centain period.
3. Chainlink VRF should generate a provably random number.
4. Chainlink Automation should trigger the lottery draw regularly.
